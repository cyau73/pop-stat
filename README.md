Welcome to the Population Statistics Dashboard

The dashboard displays countries and their total citizen and non-citizen percentages.
Data is obtained from verified published sources from the country's govt data eiher from API or manual entry.
Breakdown of the non-citizen categories will be provided based on avaiablilty from the govt data source.

We will add more countries based on our availability to work on htis project. Any help will be greatly apprecaited.

The code is based on data provided by SingStat https://www.singstat.gov.sg/

The dashboard is avilaable at https://pop-stat.vercel.app/
